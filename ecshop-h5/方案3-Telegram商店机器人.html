方案三：Telegram AI 商店机器人

用 Telegram bot 做 AI 购物助手 — 客户直接在 Telegram 上聊天下单

核心流程

  • • 客户添加 @ECShopBot 开始对话
  • • AI 导购：欢迎 + 引导描述需求
  • • 客户发文字：想买什么
  • • AI 搜索商品 → 返回图文卡片
  • • 内联按钮："加购物车" "查看详情"
  • • 客户点击即可操作，无需打字
  • • AI 汇总购物车 → 询问确认下单
  • • 客户确认 → 订单生成 → 通知

技术栈

  • • 消息层：Hermes Agent Telegram Gateway（已有）
  • • 新增：ECShop Hermes Skill
  • • Skill Tools：shop API tools
  • • LLM：MiniMax / DeepSeek
  • • 商品展示：Telegram 图文消息
  • • 交互：Inline Keyboard 按钮
  • • 订单：对接 ECShop API
  • • 部署：新增 profile bot5
  • • 看门狗：纳入现有 watchdog

优缺点

  • ✅ 利用已有 Hermes 基础设施
  • ✅ 最快落地（加个 Skill 即可）
  • ✅ 天然支持图文 + 按钮交互
  • ✅ 独立看门狗，高可用
  • ✅ 适合海外/华人 Telegram 用户
  • ⚠️ 需客户有 Telegram 账号
  • ⚠️ 不适合纯国内客户群体
  • ⚠️ 商品展示受限于 Telegram 消息
