## 📊 数据流程

```
1. 产品进入传送带
      ▼
2. 相机1扫描条码 → 边缘识别(10ms)
      ▼
3. 相机2-4拍照 → 边缘推理(50ms)
      ▼
4. 正常商品 → 直接分拣命令
5. 异常/未知 → 云端大模型(500ms)
      ▼
6. 生成抓取指令
      ▼
7. 机械臂执行 → 放置到对应料箱
```

# 🤖 Piper机械臂产品规格

## 产品简介
Piper是由Seeed Studio推出的桌面级机械臂，基于Raspberry Pi CM4控制，6自由度协作臂。

## 规格参数
| 参数 | 规格 |
|------|------|
| 自由度 | 6 DOF |
| 控制主板 | Raspberry Pi CM4 |
| 接口 | Ethernet / WiFi |
| 通信协议 | Modbus TCP / ROS |
| 夹爪 | 电动夹爪 |
| 有效载荷 | 1kg |
| 工作半径 | ~400mm |
| 电源 | 12V/5A DC |

## 控制方式
- Python SDK (pypiper)
- ROS/ROS2
- Blockly图形化编程
- REST API

# 💻 控制代码示例

## 1. Piper 机械臂控制
```python
import pypiper

arm = pypiper.Piper("/dev/ttyUSB0")

# 移动到抓取位置
arm.move_to([15.2, -8.5, 22.0, 45.0, 0, 90])

# 抓取
arm.gripper_close()
arm.sleep(0.5)

# 移动到放置位置
arm.move_to([30.0, 10.0, 15.0, -45.0, 0, -90])

# 放置
arm.gripper_open()
```

## 2. DABAI DC1 图像获取
```python
import requests

# 获取双目图像
response = requests.post("http://dc1.local:8080/infer", json={
    "image": "stream",
    "models": ["barcode", "object"]
})

data = response.json()
print(data["barcode"])  # 条码识别结果
print(data["objects"])   # 物体检测结果
```

## 3. 云端大模型决策
```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-xxx",
```
