方案一：嵌入式 AI 购物助手

在 ECShop 页面嵌入 AI 对话窗口 — 客户边逛边聊，AI 导购推荐并协助下单

核心流程

  • • 客户打开 ECShop 商城页面
  • • 右下角出现 AI 助手浮窗按钮
  • • 点击后弹出聊天窗口
  • • 客户自然语言描述需求：
  •   "给我推荐一款德国奶粉"
  •   "这个商品有优惠吗？"
  •   "帮我下单买 2 罐"
  • • AI Agent 通过 Function Call 操作商城
  • • 搜索商品 → 加购物车 → 生成订单
  • • 客户确认后直接下单

技术栈

  • • 前端：JS 浮窗组件（vanilla / React）
  • • 通信：WebSocket 实时双向通信
  • • AI 代理：Python FastAPI 服务
  • • LLM：MiniMax 或 DeepSeek
  • • Function Calling：tool-based agent
  • • 对话记忆：Redis / SQLite
  • • 对接：ECShop Lumen API
  • • WebSocket 库：`python-socketio`
  • • 部署：Docker 容器（现有环境）

优缺点

  • ✅ 客户无感接入，不用离开商城
  • ✅ 沉浸式购物体验，边逛边聊
  • ✅ 降低客户决策成本，提高转化率
  • ✅ 可收集对话数据优化推荐
  • ⚠️ 需嵌入 ECShop 页面（改 PHP 模板）
  • ⚠️ 实时 WebSocket 增加服务器开销
  • ⚠️ 移动端适配需要额外工作
